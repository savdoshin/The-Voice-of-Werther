# Голос Вертеры • Сократ
Одностраничное веб-приложение — персональный AI-тренер в образе Сократа для лидеров сетевой компании **«Вертера»**.
Приложение помогает лидерам через сократовские вопросы переходить от «ужасной модели» мышления к «идеальной» — с акцентом на выполнимость, экономику и системный рост.

---

## Как использовать

Приложение работает по трём основным сценариям:

1. **Разбор своей ситуации**
2. **Работа с цитатами партнёров**
3. **Работа с сомнениями по цифрам и выполнимости**

Сократ задаёт только **один вопрос за раз**, использует 5 линз анализа и помогает прийти к инсайту + конкретному действию.

## Развёртывание на GitHub Pages

1. Создай новый репозиторий на GitHub
2. Загрузи все файлы
3. Включи **GitHub Pages**:
   - Перейди в `Settings` → `Pages`
   - Выбери ветку `main` → Save
4. Приложение будет доступно по адресу:  The-Voice-of-Werther
 👉 [Перейти к Голосу Вертеры](https://savdoshin.github.io/The-Voice-of-Werther/index.html)
